Andrey Kudinov

Topological modal logic with difference modality of the real line

We give a characterization property of a Kripke frame for which there is a bounded morphism from the real line onto a frame iff this frame satisfies this property. We show how using this property prove that the logic of the real line is not finitely axiomatizable. Then we define a filtration of a model on the real line such that the frame of the filtrated model satisfies the characterization property which gives us fmp for the logic of the real line.
